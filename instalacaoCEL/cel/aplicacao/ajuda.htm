Entrando em um projeto:

Para entrar em um projeto, basta selecionar no canto superior direito da tela o projeto desejado. Após a seleção do mesmo será informado por uma mensagem na parte superior da tela se você é o administrador do projeto(o * depois do nome também serve como referência).

 

Adicionando um projeto:

Clique em "Adicionar Projeto" e entre com seu nome e descrição. Após isso um projeto vazio estará disponível no menu de seleção (caso ele não esteja visível, clique em "atualizar"). O criador do projeto terá automaticamente o status de administrador do projeto.

 

Removendo um projeto:

Clique em "Remover Projeto" e posteriormente em "Apagar Projeto". Todos os dados do projeto serão apagados(cenário, léxico, participantes, projetos gerados, etc).

CUIDADO!O projeto será completamente apagado, sem possibilidade de retorno. Todos os outros membros do projeto não terão mais acesso ao mesmo!

Essa opção só está disponível para p administrador do projeto.

 

Alterarando o cadastro:

É importante que os dados cadastrais sejam sempre atualizados. Para isso basta clicar em "Alterar Cadastro"e fazer as alterações necessárias, clicando a seguir em "atualizar".

 

Adicionando um cenário:

Clique na opção "Adicionar Cenário" e preencha os campos da mesma, após o término do preenchimento, caso o usuário seja o administrador do projeto, o cenário será automaticamente incluido neste(caso ele não esteja visível, clique em atualizar no canto superior direito da tela). Se o usuário não for o administrador, será enviado uma mensagem para o administrador do projeto falando que existe um novo cenário aguardando a sua aprovação.

Para que o cenário seja adiconado é necessário que pelo menos os campos título, ojetivo e contexto sejam preenchidos. Caso algum não seja preenchido, o sistema o avisará.

Quando um cenário é adicionado com sucesso ao projeto, este aparece lista no canto esquerdo da tela(caso ele não esteja visível, clique em atualizar no canto superior direito da tela).

 

Adicionando um léxico:

Clique na opção "Adicionar Léxico" e preencha os campos da mesma, após o término do preenchimento, caso o usuário seja o administrador do projeto, o léxico será automaticamente incluido neste(caso ele não esteja visível, clique em atualizar no canto superior direito da tela) e os cenários e léxicos já estarão atualizados para refletir os links com novo item do léxico. Se o usuário não for o administrador, será enviado uma mensagem para o administradore falando que existe um novo léxico aguardando a sua aprovação.

Para que o léxico seja adiconado é necessário que pelo menos os campos nome e noção sejam preenchidos. Caso algum não seja preenchido, o sistema o avisará.

Quando um léxico é adicionado com sucesso ao projeto, este aparece lista no canto esquerdo da tela(caso ele não esteja visível, clique em atualizar no canto superior direito da tela).

 

Alterar um cenário ou léxico:

Para realizar a alteração, basta selecionar o cenário ou léxico desejado na parte esquerda da tela e clicar na opção de alteração. Caso o usuário seja o administrador do sistema, a ação será executada imediatamente, caso contrário será enviado uma mensagem ao administradore relatando o pedido de alteração.

 

Apagar um cenário ou léxico:

O procedimento para se apagar um cenário ou léxico é praticamente idêntico a alteração, a única diferença é que independentemente do usuário ser o administrador do projeto ou não, será necessário que a remoção seja aprovada por ele(é uma forma de se garantir que a remoção é realmente desejada).

 

Sair do sistema:

Caso deseje sair do sistema, basta clicar em "Sair" que serão fornecidas três opções:

  • Entrar novamente - Exibe novamente a tela de entrada do sistema.
  • Página inicial - Vai para a página inicial do C&L.
  • Fechar - Fecha o navegador

 

Informações do projeto:

Clique em "Info", caso o usuário não seja o administrador do projeto, serão apresentados apenas alguns dados sobre o mesmo (nome, data de criação e descrição). Caso o usuário seja o administrador, será apresentado um menu com as seguintes opções do projeto.

  • Verificar pedidos de alteração de Cenários
  • Verificar pedidos de alteração de termos do Léxico
  • Adicionar usuário (não existente) neste projeto
  • Relacionar usuários já existentes com este projeto
  • Gerar XML deste projeto
  • Recuperar XML deste projeto

 

Verificar pedidos de alteração de Cenários e Léxico:

Esta opção é exclusiva do administrador do projeto e serve para a verificação de qualquer alteração que se deseje fazer nos léxicos ou cenários do projeto. Isso inclui remoção, alteração e inclusão. Será apresentado uma lista de todas as solicitações, cabe ao administrador do projeto aprovar ou remover a mensagem da lista. Caso a solicitação seja aprovada, o sistema realizará a tarefa requisitada, mas a mensagem continuará na lista(agora com o status de aprovado), para tirar a mensagem, basta selecionar remover.

Caso a alteração ou inclusão seja feita pelo administrador, não será necessário se ter a aprovação, logo nenhuma mensagem será enviada para a lista. No caso da remoção, a mensagem será enviada e a aprovação esperada.

 

Adicionar usuário (não existente) neste projeto:

Esta opção é exclusiva do administrador do projeto e inclu no projetoi um usuário que não esteja cadastrado no sistema. Para isso basta preencher o cadastro apresentado e o usuário estará automaticamente incluído no sistema e no projeto.

 

Relacionar usuários já existentes com este projeto:

Esta opção é exclusiva do administrador do projeto e tem a função de incluir no projeto um ou mais usuários já cadastrados no sistema. O projeto estará automaticamente incluído na lista destes usuários.

 

Gerar XML deste projeto:

Esta opção é exclusiva do administrador do projeto e tem a função de gerar uma versão em XML do projeto formatada ou não. É necessário entrar a versão do projeto, esta versão pode ser numérica ou não.

A versão gerada será automaticamente salva e poderá ser recuperada mais tarde(a não ser que o projeto seja apagado).

 

Recuperar XML deste projeto:

Esta opção é exclusiva do administrador do projeto e tem como finalidade recuperar os projetos salvos em versão XML. É possível também apagar versões antigas.

 

Como participar do projeto C&L

O projeto C&L é um projeto de Software Livre, todo o código fonte está disponível para download e o envio de melhorias não é apenas aceito, como também é estimulado. Se tiver encontrado algum bug, tiver alguma sugestão ou quiser enviar novos códigos para serem incluídos no C&L, basta enviar um e-mail para per@les.inf.puc-rio.br
